Welcome to the DjangoCSV

Online service for generating CSV files with fake(dummy) data using Python and Django

To start using app visit Schemas page

Than create new Schema (or use already generated)

When creating Schema fill up name, choose separators type and create as many columns as you wish

After Schema creation visit it for next data generation

Enter the desired number of lines and click Generate. Python will send task to Celery and AJAX will ask server every second for result
